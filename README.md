Это действующий Телеграмм-бот. 

Его основные задачи:
+ Отправлять уведомления о вскоре просроченных точках проекта
+ Отправлять/изменять письма в гугл таблицах

Файлы проекта:
+ main - основной скрипт телеграмм бота (aiogram)
С помощью неё осуществляется взаимодействие между скриптами по взаимодействию с гугл-таблицами и телеграммом
+ google_scripts - функции для запросов в гугл-таблицы, выгрузка и загрузка данных в них 
+ google_requests - функции, которые разбивают входящие письма на составляющие (текст, срок, номер).
Предшествует загрузке данных в гугл таблицу.
+ send_notification - отвечает за постановку времени и отправку уведомления через телеграмм бота
+ form_notification - отвечает за формирование уведомления из загруженных таблиц
+ excel_scripts - отвечает за работу с excel таблицами - выгрузка пользователей, фильтрацию данных.
Также здесь находится функция отвечающая за поиск новейшего файла в директории и удалении остальных.
